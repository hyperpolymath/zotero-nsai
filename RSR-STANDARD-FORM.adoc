== Overview

The Rhodium Standard Repository (RSR) form defines the canonical structure for all Hyperpolymath projects. This document specifies required files, formats, and tooling.

== File Format Requirements

[cols="2,1,2"]
|===
|File |Format |Rationale

|Documentation (general)
|AsciiDoc (.adoc)
|Rich semantic markup, Zola/pandoc compatible

|LICENSE
|Plain text (.txt)
|Legal convention

|SECURITY.md
|Markdown
|GitHub security advisory integration

|CONTRIBUTING.md OR CODE_OF_CONDUCT.md
|Markdown
|One may be .adoc, one must be .md for GitHub

|FUNDING.yml
|YAML
|GitHub Sponsors integration

|STATE.scm
|Scheme (Guile)
|Homoiconic checkpoint/restore

|Configuration
|CUE (.cue)
|Type-safe config generation
|===

== Required Tooling

=== Documentation Pipeline
- **Pandoc**: Document conversion (adoc → md → html → pdf)
- **AsciiDoctor**: AsciiDoc rendering
- **Zola**: Static site generation

=== Quality Assurance
- **hunspell**: Spell checking
- **agrep**: Approximate pattern matching
- **languagetool**: Grammar checking (optional)

=== Internationalization
- **i18n**: Via locales/*/messages.json
- **gettext**: For Guile scripts

=== Package Management
- **Guix**: Primary (channels.scm)
- **Nix**: Secondary (flake.nix)
- **Deno**: Runtime (no npm)

== Licensing

=== Legal License
- **AGPL-3.0-or-later**: Default for applications
- **MIT**: For libraries/templates

=== Philosophical License
- **Palimpsest v0.8**: AI training ethics overlay

=== Contributor Framework
- **TPCF**: Tri-Perimeter Contributor Framework
  * Inner: Core maintainers
  * Middle: Regular contributors
  * Outer: Community/forks

== Language Policy

=== Accepted
[source]
----
ReScript     (.res, .resi)   # Primary application language
Guile/Scheme (.scm)          # Tooling, config, STATE
Elixir       (.ex, .exs)     # Backend services
Rust         (.rs)           # WASM, performance-critical
Lean 4       (.lean)         # Formal verification
Julia        (.jl)           # Statistical analysis
SCSS         (.scss)         # Styles
CUE          (.cue)          # Configuration
Shell        (.sh)           # Basic scripts (prefer Guile)
----

=== Banned
[source]
----
TypeScript   (.ts, .tsx)     # Use ReScript instead
CoffeeScript (.coffee)       # Obsolete
Go           (.go)           # Not in stack
npm          (package.json)  # Use Deno
Python       (.py)           # Except Salt-related
----

== Consent-Aware HTTP Compliance

All repositories MUST include `.well-known/aibdp.json` per the AI Boundary Declaration Protocol (draft-jewell-aibdp).

Required policies:
- `training`: Declare AI training consent
- `indexing`: Search engine permissions
- `generation`: Derivative work permissions
- `embedding`: Vector database permissions
