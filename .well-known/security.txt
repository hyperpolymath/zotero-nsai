# Vulnerability Reporting
# Please report security vulnerabilities privately through GitHub Security Advisories
# or by email. Do not open public issues for security vulnerabilities.

# Response Timeline
# - Initial response: 48 hours
# - Status update: 7 days
# - Fix timeline: 7-90 days (depending on severity)

# Scope
# All code in this repository is in scope for security research.
# We follow coordinated disclosure with a 90-day disclosure timeline.

# Out of Scope
# - Social engineering attacks
# - Denial of Service (DoS/DDoS)
# - Issues in dependencies (report to upstream)
# - Issues requiring physical access to user's machine
